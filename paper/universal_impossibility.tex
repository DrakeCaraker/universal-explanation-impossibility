\begin{abstract}
%%=======================================================================
We prove a universal impossibility: no explanation system can simultaneously be
faithful (reflecting the chosen hypothesis), stable (consistent across
equivalent hypotheses), and decisive (resolving all explanation queries) when
the underlying inference problem is underspecified---that is, when multiple
hypotheses fit the data equally well.
This single abstract theorem, which we call the \emph{Universal Explanation
Impossibility}, is a meta-theorem that unifies six apparently unrelated
impossibility results from across machine learning: (1)~the feature attribution
impossibility under collinearity \citep{bilodeau2024impossibility}, (2)~the
SHAP instability under model multiplicity \citep{damour2022underspecification},
(3)~the fairness metric incompatibility \citep{chouldechova2017fair,kleinberg2017inherent},
(4)~the model selection instability under the Rashomon effect
\citep{rudin2024amazing,fisher2019models}, (5)~the causal discovery ambiguity
within a Markov equivalence class \citep{verma1991equivalence}, and (6)~the
impossibility of faithful explanations for overparameterized neural networks.
In each case, underspecification induces a Rashomon property on explanation
targets, and the same four-line proof closes the impossibility.
The resolution is uniform: $G$-invariant explanation systems that aggregate
over equivalent hypotheses restore faithfulness and stability at the cost of
decisiveness, and \DASH{} (\textbf{D}iversified \textbf{A}ggregation of
\textbf{SH}AP) is a concrete instance of this resolution for feature attribution.
The impossibility is ubiquitous: any learning problem with strictly more
parameters than constraints admits a Rashomon set of positive measure, and
neural networks with hidden dimension exceeding the data dimension always fall
into this regime.
The framework is mechanically verified in Lean~4 (67~files, 319~theorems,
60~axioms, 0~\texttt{sorry}).
\end{abstract}

\section{The Abstract Framework}
\label{sec:framework}
%%=======================================================================

We introduce a minimal abstract framework sufficient to state and prove the
Universal Explanation Impossibility.  All six instances in
Section~\ref{sec:instances} are special cases.

\subsection{Explanation Systems}

\begin{definition}[Explanation System]
\label{def:explanation-system}
An \emph{explanation system} is a tuple $\mathcal{S} = (\Params, \sH, Y, \mathsf{obs}, \mathsf{exp}, \incomp)$ where:
\begin{itemize}
  \item $\Params$ is a \emph{parameter space} (the set of possible ground truths
    or data-generating processes);
  \item $\sH$ is a \emph{hypothesis space} (the set of fitted models or
    inferred structures); each $h \in \sH$ is associated with a parameter
    $\theta(h) \in \Params$ via the \emph{attribution map} $\theta: \sH \to \Params$;
  \item $Y$ is an \emph{explanation target space} (e.g., feature rankings,
    causal graphs, fairness certificates);
  \item $\mathsf{obs}: \sH \to Y$ is the \emph{observation map}, assigning
    to each hypothesis the explanation it ``honestly reports'';
  \item $\mathsf{exp}: \sH \to Y$ is the \emph{explanation map}, the
    procedure under evaluation; and
  \item $\incomp\;\subseteq Y \times Y$ is an \emph{incompatibility relation}
    on explanations (e.g., strict ordering $j \succ k$ vs.\ $k \succ j$,
    or two mutually exclusive fairness certificates).
\end{itemize}
\end{definition}

The explanation map $\mathsf{exp}$ is what a practitioner \emph{chooses} (e.g.,
SHAP, LIME, a fixed causal orientation rule); the observation map
$\mathsf{obs}$ records what the hypothesis \emph{actually implies}.  The
incompatibility relation captures when two explanations cannot both be
simultaneously correct.

\subsection{The Three Desiderata}

\begin{definition}[Faithful]
\label{def:faithful}
An explanation map $\mathsf{exp}$ is \emph{faithful} if for every hypothesis
$h \in \sH$,
\[
  \mathsf{exp}(h) = \mathsf{obs}(h).
\]
That is, the produced explanation agrees with what the hypothesis itself
implies.
\end{definition}

\begin{definition}[Stable]
\label{def:stable}
An explanation map $\mathsf{exp}$ is \emph{stable} if for every pair of
hypotheses $h, h' \in \sH$ with $\theta(h) = \theta(h')$,
\[
  \mathsf{exp}(h) = \mathsf{exp}(h').
\]
That is, equivalent hypotheses (same ground-truth parameter) receive the same
explanation.
\end{definition}

\begin{definition}[Decisive]
\label{def:decisive}
An explanation map $\mathsf{exp}$ is \emph{decisive} if for every hypothesis
$h \in \sH$, the explanation $\mathsf{exp}(h)$ is \emph{resolved}: it contains
no undecided or tied queries.  Formally, there is no $y \in Y$ such that both
$(\mathsf{exp}(h), y) \in \incomp$ and $\mathsf{exp}(h) = y$.
\end{definition}

In the feature attribution instance (Section~\ref{sec:instance-attribution}),
faithfulness is faithfulness to the model's own SHAP values, stability is
consistency across retraining seeds, and decisiveness is the requirement to
rank every feature pair strictly (no ties).

\subsection{The Rashomon Property}

The key structural condition linking underspecification to explanation
impossibility is the following.

\begin{definition}[Rashomon Property]
\label{def:rashomon}
An explanation system $\mathcal{S}$ has the \emph{Rashomon property} if for
every resolved pair of incompatible explanations $(y, y') \in \incomp$ with
$y \neq y'$, there exist hypotheses $h, h' \in \sH$ with
$\theta(h) = \theta(h')$ such that
\[
  \mathsf{obs}(h) = y \quad \text{and} \quad \mathsf{obs}(h') = y'.
\]
That is, for every incompatible explanation pair, two hypotheses with the same
ground-truth parameter exist that \emph{honestly} report the respective
incompatible explanations.
\end{definition}

The Rashomon property is the formal statement of underspecification at the
explanation level: the data do not uniquely determine the explanation, even in
principle.  It generalizes the Rashomon effect of
\citet{rudin2024amazing,fisher2019models} from model selection to arbitrary
explanation targets.

\subsection{Main Theorem}

\begin{theorem}[Universal Explanation Impossibility]
\label{thm:universal}
If an explanation system $\mathcal{S} = (\Params, \sH, Y, \mathsf{obs}, \mathsf{exp}, \incomp)$
has the Rashomon property, then no explanation map $\mathsf{exp}$ can
simultaneously be faithful, stable, and decisive.
\end{theorem}

\begin{proof}
By the Rashomon property, there exist incompatible explanations $y \neq y'$
with $(y, y') \in \incomp$ and hypotheses $h, h'$ satisfying
$\theta(h) = \theta(h')$, $\mathsf{obs}(h) = y$, and $\mathsf{obs}(h') = y'$.

Suppose for contradiction that $\mathsf{exp}$ is faithful, stable, and
decisive.  By faithfulness, $\mathsf{exp}(h) = \mathsf{obs}(h) = y$ and
$\mathsf{exp}(h') = \mathsf{obs}(h') = y'$.  But by stability and
$\theta(h) = \theta(h')$, we must have $\mathsf{exp}(h) = \mathsf{exp}(h')$,
i.e., $y = y'$.  This contradicts $y \neq y'$.
\end{proof}

\begin{remark}
The proof uses decisiveness only implicitly, through the requirement that
$\mathsf{obs}(h)$ is a resolved, specific explanation (not a set or
distribution).  The impossibility persists whenever $\mathsf{obs}$ is
single-valued and the incompatibility relation is asymmetric.
\end{remark}

\begin{remark}
The theorem is deliberately parameter-free: it requires no regularity
conditions on $\sH$, no metric structure on $Y$, and no probabilistic
assumptions.  All quantitative depth (divergence rates, sample complexity,
ensemble sizes) appears in the instance-specific sections below.
\end{remark}

\begin{corollary}[Decidable Underspecification Implies Impossible Explanation]
\label{cor:decidable}
If the data-generating process $\theta$ is not identifiable from the
observations---i.e., if there exist $h, h'$ with $\theta(h) = \theta(h')$ yet
$\mathsf{obs}(h) \neq \mathsf{obs}(h')$---then the Rashomon property holds
whenever the explanation targets are ordered by an incompatibility relation
with at least one incompatible pair.
\end{corollary}

\section{Lean Formalization}
\label{sec:lean}

verified in Lean~4 \citep{demoura2021lean4} using Mathlib
\citep{mathlib2020}.  Table~\ref{tab:lean-summary} summarizes the
formalization.

\centering

\label{tab:lean-summary}

Component & Count & Notes \\

\bottomrule
\end{tabular}
\end{table}

The core impossibility (\texttt{universal\_explanation\_impossibility} in
\texttt{UniversalImpossibility/Universal.lean}) depends on zero axioms beyond
the Rashomon property, which is stated as a hypothesis rather than an axiom.
Each instance introduces the minimum axioms needed to verify that its
particular explanation system satisfies the Rashomon property.

The Lean proof of Theorem~\ref{thm:universal} is four lines, mirroring the
paper proof exactly.  The formalization also includes:
\begin{itemize}
  \item The $G$-invariant resolution (Section~\ref{sec:resolution}):
    \texttt{Corollary.lean}, \texttt{DesignSpace.lean};
  \item Quantitative bounds for GBDT (\texttt{Ratio.lean}), Lasso
    (\texttt{Lasso.lean}), and neural networks (\texttt{NeuralNet.lean});
  \item The fairness instance (\texttt{FairnessAudit.lean}) and causal
    instance (\texttt{CausalDiscovery.lean}).
\end{itemize}
